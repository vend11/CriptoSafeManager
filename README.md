# CryptoSafe Manager 🛡️

Это приложение — персональный сейф для твоих паролей и логинов. Главная цель проекта — не просто хранить данные, а делать это максимально безопасно, используя современные подходы к защите информации.

### Что умеет приложение:

* **Первая настройка**: При самом первом запуске открывается «Мастер настройки». Он помогает создать твой главный (мастер) пароль и выбрать место на компьютере, где будет лежать зашифрованный файл с данными.
* **Удобное управление**: В главном окне ты можешь создавать, изменять и удалять записи. Для каждой записи можно указать название сервиса, логин, адрес сайта и теги для быстрого поиска.
* **Скрытые пароли**: В формах ввода есть кнопка-глазик. Пароль скрыт звездочками, пока ты сам не захочешь его увидеть.
* **Умная таблица**: Все твои данные отображаются в аккуратном списке, где их легко просматривать и редактировать.

### Как обеспечивается безопасность:

* **Защита памяти**: Приложение очень осторожно работает с оперативной памятью. Как только пароль тебе больше не нужен, программа «затирает» его в памяти компьютера.
* **Шифрование**: Все данные перед сохранением в базу превращаются в нечитаемый код. Даже если кто-то украдет файл базы данных, он не увидит там твоих паролей.
* **Контроль действий**: Приложение фиксирует каждое важное событие (например, добавление или удаление пароля). Это нужно для того, чтобы всегда можно было проверить историю изменений.
* **Резервные копии**: В меню есть функция создания копии твоей базы данных, чтобы ты не потерял доступ к аккаунтам, если что-то случится с основным файлом.

Проект делится на 8 спринтов, каждый из которых расширяет функциональность:

Спринт 1 — Технический фундамент: модульная архитектура, зашифрованная база данных, каркас GUI.

Спринт 2 — Менеджер ключей и расширения: ключи шифрования, безопасное хранение, конфигурации.

Спринт 3 — Реализация криптографии: полноценный AES-GCM, замена заглушек.

Спринт 4 — Продвинутый интерфейс: буфер обмена, авто-блокировка, фильтры.

Спринт 5 — Журнал аудита и события: логирование действий, экспорты.

Спринт 6 — Пользовательские настройки: темы, языки, удобство.

Спринт 7 — Безопасность и тестирование: тестовый охват, CI/CD.

Спринт 8 — Деплой и документация: финальная сборка, установки, бэкапы.

Инструкция по созданию виртуального окружения
# 1. Клонируем репозиторий
git clone https://github.com/your_username/cryptosafe-manager.git
cd cryptosafe-manager

# 2. Создаём виртуальное окружение
python -m venv venv

# 3. Активируем его
# Windows
venv\Scripts\activate

# macOS/Linux
source venv/bin/activate

# 4. Устанавливаем зависимости
pip install -r requirements.txt
